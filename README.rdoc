== README

Para rodar esse código siga os seguintes passos:

* Instalar ruby 2.2

* Rodar o comando bundle para instalar as dependências

* rake db:create, rake db:migrate, rake db:seed (Cria o banco, cria estrutura e popula o banco com usuário inicial)

* rails s para rodar o aplicativo e acessar a url indicada no console, provavelmente http://localhost:3000

* rake spec para rodar os testes

* O usuário padrão é admin@admin.com e a senha admin123
